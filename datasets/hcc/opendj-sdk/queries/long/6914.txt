When you setup server in interactive mode there is `Equivalent non-interactive command:` at the end of setup. But the command is not equivalent, it is missing `acceptLicense` argument.

steps to reproduce:

	run ./setup
	answer all prompts
	don't setup server at the end of setup - answer 3) Quit
	try to use non-interactive command that was created at the end of setup
	command will fail with:


You must accept the product license to run setup. Either use interactive mode
or the 'acceptLicense' argument
