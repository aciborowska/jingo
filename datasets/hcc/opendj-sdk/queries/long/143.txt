Please note that Ubuntu 10 now does not deliver Sun JDK as part of their repository, but only OpenJDK which seems to work (as far as setup script is concerned). To use the Sun JDK I have performed the following steps:

	download the self-extracting binary of JDK from the JDK site
	unpacked into /usr/jdk
	set the environment variables as follows:



export JAVA_HOME=/usr/jdk/jdk1.6.0_25
export PATH=$JAVA_HOME/bin:$PATH
