Tool usage includes flags after search filter which are misinterpreted as attribute names. Same applies to dev guide tool reference.


Usage:  searchrate  {options} [filter format string] [attributes ...]

...

Example:

  searchrate -p 1389 -b "dc=example,dc=com" \
    -g "rand(names.txt)" "(uid=%s)" -A -F -c 4 -t 4
