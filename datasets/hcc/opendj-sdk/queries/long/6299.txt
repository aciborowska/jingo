The -XX:+UnlockDiagnosticVMOptions parameter is required when using the -XX:+UseAES -XX:+UseAESIntrinsics options when using Java 11.

 

$ ./jdk-11.0.3/bin/java -XX:+UseAES -XX:+UseAESIntrinsics -versionError: VM option 'UseAESIntrinsics' is diagnostic and must be enabled via -XX:+UnlockDiagnosticVMOptions.
Error: The unlock option must precede 'UseAESIntrinsics'.
Error: Could not create the Java Virtual Machine.
Error: A fatal exception has occurred. Program will exit.
