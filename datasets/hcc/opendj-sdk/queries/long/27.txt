But adding over LDAP or in a schema file fails when there's an extension like X-SCHEMA-FILE '00-core.ldif'

changetype: modify
add: ldapsyntaxes
ldapsyntaxes: ( 1.1.1 DESC 'test syntax' X-ENUM ( 'test' 'enum' ) X-SCHEMA-FILE '97-test.ldif' )
