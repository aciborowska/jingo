See:  OPENDJ-1082 Order attributes according to search request attribute list order

Based on the above RFE, the order of the return attributes was sent in the order they were requested.  There is a regression in the behavior starting with version 6.5.0.

Test: request attributes in the following order

	mail givenName cn postalAddress
