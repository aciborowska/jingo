When running dsconfig in interactive mode, there is no option to set an enum property to an expression. This is exactly the same issue as OPENDJ-5291 but for enum.

Enums are used for the index-type for example:


dsconfig set-backend-index-prop \
          --backend-name userRoot \
          --index-name cn \
          --offline


then the menu is:


Select the values you wish to add:

    1)  extensible
    2)  ordering
    3)  presence
    4)  Add all values

    q)  quit
    c)  cancel
    ?)  help


There is no way to add an expression
