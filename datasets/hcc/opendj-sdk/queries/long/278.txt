Create the following PWP:

dn: cn=Subtree Password Policy,ou=people,dc=ldap,dc=com
objectClass: pwdPolicy
objectClass: ldapSubentry
objectClass: top
pwdFailureCountInterval: 300
pwdMaxFailure: 3
pwdAttribute: userPassword
pwdAllowUserChange: TRUE
cn: Subtree Password Policy
pwdLockoutDuration: 300
pwdLockout: TRUE

Then check a subordinate user's entry:

ldapsearch -D "cn=Directory Manager" -w password -b ou=people,dc=ldap,dc=com "uid=testuser" pwdPolicySubentry
dn: uid=testuser,ou=people,dc=ldap,dc=com
pwdPolicySubentry: cn=Default Password Policy,cn=Password Policies,cn=config

It should have inherited the sub-entry PWP, not the default.
