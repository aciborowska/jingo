The connected-to monitoring attribute has inconsistent formatting which depends on the type of replication node. In particular:


	local DS:  ubuntu-1-private.example.com/192.168.56.2:8989
	DS connected to local RS: Replication Server 8989 7364
	DS connected to remote RS: Connected replication server RS(20750) ubuntu-2-private.example.com:8989,cn=Replication server RS(7364) ubuntu-1-private.example.com:8989,cn=dc_example_dc_com,cn=Replication
