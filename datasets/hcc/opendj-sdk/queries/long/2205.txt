It appears, that the loop that should select files to be deleted to free disk space will never run because of incorrect loop condition:

for (int j = files.length - 1; j < 1; j--)

(Line 138 in trunk/opendj/opendj-server-legacy/src/main/java/org/opends/server/loggers/FreeDiskSpaceRetentionPolicy.java, 
http://sources.forgerock.org/browse/opendj/trunk/opendj/opendj-server-legacy/src/main/java/org/opends/server/loggers/FreeDiskSpaceRetentionPolicy.java?hb=true#to138 )

The obvious fix is that j < 1 should be replaced with j > 1.
