In https://backstage.forgerock.com/docs/ds/6.5/getting-started/#try-repl the example uses the wrong port for the first server.

It should be 1389. 11389 is the LDAP port on the second server:


# On the first server, read the description "Replicate this" to see the effects of your change:
ldapsearch \
 --hostname localhost \
 --port 11389 \
 --baseDn dc=example,dc=com \
 "(cn=Abagail Abadines)" \
 description

# On the second server, read the description "Replicate this" to check that the change has been replicated:
ldapsearch \
 --hostname localhost \
 --port 11389 \
 --baseDn dc=example,dc=com \
 "(cn=Abagail Abadines)" \
 description
