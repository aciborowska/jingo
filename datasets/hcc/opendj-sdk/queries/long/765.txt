If you have a single-valued attribute (e.g. defaultserverlist) then the following modify op:


dn: uid=test,dc=example,dc=com
changetype: modify
replace: defaultserverlist
defaultserverlist: original
-
delete: defaultserverlist
-



will result in an entry with no defaultserverlist attribute (good) but ds-sync-hist doesn't record the delete:


ds-sync-hist: defaultserverlist:csn:repl:original



This can cause problems when trying to add a new value.
