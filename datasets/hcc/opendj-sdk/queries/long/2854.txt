The AVA class should expose a getter method for obtaining the original user provided attribute type. If an AVA is parsed using valueOf() then the getter should return the type in string passed to valueOf(). For example, assuming the method's name is getAttributeName():



AVA ava1 = AVA.valueOf("CN=value");
AVA ava2 = AVA.valueOf("commonname=value");

// CN and commonname map to the same AttributeType
assertThat(ava1.getAttributeType()).isEqualTo(ava2.getAttributeType());

// CN != commonName
assertThat(ava1.getAttributeName()).isNotEqualTo(ava2.getAttributeName());
