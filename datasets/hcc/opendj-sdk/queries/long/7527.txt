README that comes with example-plugin and example-pwdscheme do not have accurate information. They are missing the deploymentKeyPassword in setup and they point to the wrong location for the zip:


1. Unpack and install, but do not start, a directory server:

```
/path/to/opendj/setup \
    --rootUserDN "uid=admin" \
    --rootUserPassword password \
    --hostname localhost \
    --ldapPort 1389 \
    --adminConnectorPort 4444 \
    --profile ds-evaluation \
    --acceptLicense
```

2. Unzip the example plugin distribution file:

```
unzip /path/to/opendj/example-plugin.zip -d /tmp
