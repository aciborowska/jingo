The modifiersName and modifyTimestamp are not included in 99-user.ldif for the replica if the schema objects/attributes are added before enabling replication. The operational attributes are present if you add the schema objects after enabling replication. So this is a minor inconsistency.

[2] Add new schema objects before enabling replication.
