Setup DS specifying multiple profiles.
For each profile specify the same baseDn, e.g.


setup directory-server --rootUserDn "cn=directory manager" --rootUserPasswordFile PATH_TO_PWDFILE --doNotStart --acceptLicense --adminConnectorPort 5445 --ldapPort 10389 --hostname SOME_FQDN
    --profile am-cts --set am-cts/amCtsAdminPassword:SOME_PASSWORD --set am-cts/baseDn:dc=openam,dc=forgerock,dc=org
    --profile am-config --set am-config/amConfigAdminPassword:SOME_PASSWORD --set am-config/baseDn:dc=openam,dc=forgerock,dc=org
    --profile am-identity-store --set am-identity-store/amIdentityStoreAdminPassword:SOME_PASSWORD --set am-identity-store/baseDn:dc=openam,dc=forgerock,dc=org



--> setup completes successfully
