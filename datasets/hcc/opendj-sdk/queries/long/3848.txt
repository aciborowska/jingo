dsconfig tool used with either the displayCommand or the commandFilePath option prints equivalent non interactive command line to perform a configuration action.
At the moment the printed command line does not contain the connection argument such as host name, bind DN, port...

Sample below presents the output of a $ ./dsconfig --discplayCommand after modifying a backend  index:

[...]
The Backend Index was modified successfully

The equivalent non-interactive command-line is:
dsconfig set-backend-index-prop \
          --backend-name userRoot \
          --index-name name \
          --set confidentiality-enabled:false \
          --no-prompt



The equivalent non-interactive command-line should be:

dsconfig set-backend-index-prop \
          --backend-name userRoot \
          --index-name name \
          --set confidentiality-enabled:false \
          --hostname gaetan-laptop \
          --port 4444 \
          --bindDn cn=directory\ manager \
          --bindPassword ****** \
          --no-prompt
