On a topology with 2xDS and 2xRS:

1. set log publisher on DS1 with 


.\DS1\bat\dsconfig.bat -h localhost -p 4444 -D "cn=Directory Manager" -w password -X set-log-publisher-prop --publisher-name "File-Based Access Logger" --set enabled:true --set suppress-internal-operations:false --set log-format:combined --advanced -n

2. do modification on DS1:


.\DS1\opendj\bat\ldapmodify.bat -h localhost -p 1389 -D "cn=Directory Manager" -w password
dn: uid=user.8,ou=people,dc=com 
changetype: modify 
add: roomNumber 
roomNumber: 25

3. stop RS2:


.\RS2\opendj\bat\stop-ds.bat -h localhost -p 4447 -D "cn=Directory Manager" -w password -X

4. do modification on DS2:


.\DS2\opendj\bat\ldapmodify.bat -h localhost -p 1390 -D "cn=Directory Manager" -w password
dn: uid=user.9,ou=people,dc=com 
changetype: modify 
add: roomNumber 
roomNumber: 25

last command results in error:


ERROR: -- rc -- returned 53, expected 0 -- stdout -- # The LDAP modify request failed: 53 (Unwilling to Perform) # Additional Information: The Replication is configured for suffix dc=com but was not able to connect to any Replication Server
