./bin/start-ds

./bin/backup  --hostname opendj.example.com  --port 4444  --bindDN "uid=admin"  --bindPassword password  --backupAll   --backupDirectory ./bak  --start 0  --trustAll

./bin/ldapsearch -p 1389 -D "uid=admin" -w password -b "ds-backup-directory-path=./bak/appData,cn=backups" "(&)"


Actual Result


# The LDAP search request failed: 80 (Other)
# Additional Information:  An error occurred while attempting to examine the requested backup directory:  NullPointerException(BackupBackend.java:380)


Expected Result

The backup entry, or in the worst case, a more meaningful message without a NPE.
