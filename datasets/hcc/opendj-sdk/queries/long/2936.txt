If the OpenDJ is not setup for replication and the user tries to change the password for Global Administrator, a NullPointerException will be encountered. 


  ./ldappasswordmodify --port 4444 --hostname localhost --bindDN "cn=Directory Manager" --bindPassword password --authzID "cn=admin,cn=Administrators,cn=admin data" --newPassword password --useStartTLS 
Exception in thread "main" java.lang.NullPointerException
	at org.opends.server.tools.LDAPConnection.connectToHost(LDAPConnection.java:247)
	at org.opends.server.tools.LDAPPasswordModify.mainPasswordModify(LDAPPasswordModify.java:681)
	at org.opends.server.tools.LDAPPasswordModify.main(LDAPPasswordModify.java:101)



As Null Pointer Exception is not desirable in programming, the code should be enhanced to catch this error and provide a more friendly message instead.
