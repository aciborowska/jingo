If we use the setup tool with an invalid server id, it fails as expected but there is a small typo in the error message:


./DJ5/opendj/setup directory-server -h nameserver.example.com -p 1393 -D "cn=myself" -w password --adminConnectorPort 4448 -Z 1640 -b dc=com   -O --monitorUserDn uid=Monitor --monitorUserPassword password --serverId "_wrong" 	

Server ID '_wrong' in not valid. A Server ID is an ASCII alpha-numeric string
possibly containing underscore and hyphen characters.



(Note the "in" instead of "is")
