dsconfig offline does not allow to edit a config file if a server is running even if the config file being edited is not the one used by the server. 

Reproduce case: 

bin/start-ds
bin/dsconfig set-log-publisher-prop 
--publisher-name "File-Based Access Logger" 
--set enabled:true 
--offline
--configFile "/a/path/to/a/config/file/not/used/by/the/server/config.ldif"
