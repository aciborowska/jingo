Some olinks to command reference pages unhelpfully include titles when the olinks are resolved. E.g., "Use the setup in the Tools Reference command-line tool."

(Code: <olink targetdoc="tools-reference" targetptr="setup-1"><command>setup</command></olink>)

These probably need something like xrefstyle="select: nodocname". See http://www.sagehill.net/docbookxsl/CustomXrefs.html#Xrefstyle.
