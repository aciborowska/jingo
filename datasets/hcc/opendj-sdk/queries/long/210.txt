  1.4.1.1466.115.121.1.40 X-ORIGIN 'user defined' )

 UP top STRUCTURAL MUST uid MAY ( cn $ userPassword $ description $ PublicKey 
 ) X-ORIGIN 'user defined' )

Processing ADD request for cn=key,dc=example,dc=com
