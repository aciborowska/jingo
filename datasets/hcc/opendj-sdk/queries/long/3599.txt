In order to use the LDAP based keystore:

	we install DJ
	we add a ou=keystore,... entry
	we try to generate key pair using keytool, with ldap_keystore_configuration.cfg containing:

org.forgerock.opendj.security.keyStoreBaseDN=ou=keystore,dc=example,dc=com
org.forgerock.opendj.security.host=localhost
org.forgerock.opendj.security.port=1389
org.forgerock.opendj.security.bindDN=cn=myself
org.forgerock.opendj.security.bindPassword=password





we get:

keytool -genkeypair -alias "private-key" -ext "san=dns:localhost" -dname "CN=localhost,O=Example Corp,C=FR" -J-cp -J${INSTANCE1}/opendj/lib/bootstrap-client.jar -providerName OpenDJ -providerClass org.forgerock.opendj.security.OpenDjSecurityProvider -providerArg ./ldap_keystore_configuration.cfg -storetype LDAP -keystore NONE -storepass:file ${INSTANCE1}/opendj/config/ldap.pin -keypass:file ${INSTANCE1}/opendj/config/ldap.pin
keytool error: java.lang.reflect.InvocationTargetException
