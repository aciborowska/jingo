python3 run-pybot.py -v -s replication_group3.mixedTopologies -t migrate_rt1_topo DJ

	Setup old DSRS-DSRS topo
	Setup 7.0.0 DSRS and do not start it
	Configure 7.0.0 DSRS to be compatible with 6.5.3
	run dsrepl add-local-server... cmd
	start the 7.0.0 server
	run dsrepl initialize on this 7.0.0 server and initialize from 6.5.3 server
	check ldap operations works (add, mod, del users on 6.5.3 server and check those changes are propagated to 7.0.0 server)
	stop one of the old DSRS servers
	repeat steps 3,4,5,6,7 with another 7.0.0 server
	stop the remaining old DSRS server
	run dsrepl cleanup on each new server
	try to add new entry to topology
