 

Put is not allowing a null value for multiValued attributes.


curl -X PUT \
http://localhost:8080/openidm/internal/role/openidm-reg \
-H 'Content-Type: application/json' \
-H 'X-OpenIDM-Password: openidm-admin' \
-H 'X-OpenIDM-Username: openidm-admin' \
-d '{
"_id": "openidm-reg",
"name": "openidm-reg",
"description": "Anonymous access role",
"temporalConstraints": null,
"condition": null
}'



{
"code": 400,
"reason": "Bad Request",
"message": "The request cannot be processed because an array of values was not provided for the multi-valued field '/temporalConstraints'"
}
