Restoring a backup which has disabled backend throws NullPointerException.

Steps to reproduce:

	We can run with automate tests 

python3.6 run-pybot.py -t restore_one_disabled_backend_online OpenDJ

 if case that remove setup with except_instance_version


	Download OpenDJ 6.0.0 cf670d07003, unzip, setup and configure OpenDJ (navigate to OpenDJ folder)
	Create back up of data

./bin/backup -h localhost -p 4444 -D "cn=Directory Manager" -w password -X -n userRoot -d /bak/backupTestDir

	Disable userRoot backend

./bin/dsconfig -h localhost -p 4444 -D "cn=Directory Manager" -w password -X set-backend-prop --backend-name userRoot --set enabled:False --trustAll --no-prompt -n

	Execute restore online for the backup with disable backend

/bin/restore -h localhost -p 4444 -D "cn=Directory Manager" -w password -X -d /bak/backupTestDir
