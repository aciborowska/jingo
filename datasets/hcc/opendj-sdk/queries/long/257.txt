[mark@openam ~]$ /path/to/OpenDJ/bin/ldapsearch -p 1389 -h openam-bis.example.com -b dc=example,dc=com uid=bjensen description
dn: uid=bjensen,ou=People,dc=example,dc=com
description: Modified after pausing replication.


Instead, to pause replication:

$ dsconfig -p 5444 -h `hostname` -D "cn=Directory Manager" -w password set-synchronization-provider-prop --provider-name "Multimaster Synchronization" --set enabled:false -X -n

Do NOT allow data to be modified on the paused replica at this point as they are lost, causing the replica to diverge.

To enable again:
$ dsconfig -p 5444 -h `hostname` -D "cn=Directory Manager" -w password set-synchronization-provider-prop --provider-name "Multimaster Synchronization" --set enabled:true -X -n
