When using dsconfig interactively, dsconfig prompts the equivalent non-interactive command-line without the hostname, port and bindDn :

The equivalent non-interactive command-line is:
dsconfig list-log-publishers \
          --bindPassword ****** \
          --trustAll \
          --no-prompt


 
 On 5.5, the equivalent non-interactive command line is complete: 

The equivalent non-interactive command-line is:
dsconfig list-log-publishers \
          --hostname macgrady.local \
          --port 4444 \
          --bindDn cn=Directory\ Manager \
          --bindPassword ****** \
          --trustAll \
          --no-prompt
