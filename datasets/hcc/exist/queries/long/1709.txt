### What is the problem

On latest develop, when using httpclient to GET a resource which is returned as xs:base64Binary, a following util:binary-to-string will fail with a "Socket is closed" error.

### What did you expect
Sample query below works fine in eXist 3.6.1 and returns a JSON string.

### Describe how to reproduce or add a test
```xquery
xquery version "3.1";

import module namespace http="http://expath.org/ns/http-client";

let $url := "https://www.ssrq-sds-fds.ch/places-db-edit/views/loc-search.xq?query=sevelen"
let $request :=
    <http:request method="GET" href="{$url}"/>
let $response := http:send-request($request)
let $json := util:binary-to-string($response[2])
return
    $json
```
