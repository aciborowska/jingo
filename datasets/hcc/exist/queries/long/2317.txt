While accessing an out-of-bound index of array in XQuery using the parentheses notation correctly throws XQuery error, using array:get() function for the same purpose ends with an IndexOutOfBoundsException thrown within the servlet.

**Steps to reproduce:**

1.  Open new XQuery 3.1 window
2. evaluate the following expression:
```
  let $a := [ 1, 2, 3 ]
  return $a(10)
```
This correctly ends with **err:FOAY0001 Position 10 does not exist in this array.**  being reported.
3.   Change the second line of the expression to
 ` return array:get($a, 10)`
