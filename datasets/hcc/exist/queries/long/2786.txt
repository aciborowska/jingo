### What is the problem
xpath bug.  Result sets are incorrect.

first xpath is returning more than one item - ("topicValue","topicValue")
second xpath is not returning an empty set - ("topicValue")

### What did you expect
first xpath should have only returned 1 value - ("topicValue")
second xpath should have returned an empty set - ()

let $dita :=
