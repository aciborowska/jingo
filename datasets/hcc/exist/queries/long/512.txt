simplified test case that produces the NPE using current develop branch in eXide:

```
  1 < (2, 2)[1 + 2]
```

The NPE is detailed in the `exist.log` entry below.

Here are three similar expressions that produce the expected result (false) instead of the NPE:

```
  1 < (2, 2)[3]
  1 < (2, 2)[position() = 1 + 2]
  1 < (2)[3]
```

I tested all versions in Saxon and got the expected result for all (false).
