### What is the problem

The axes `preceding-sibling` and `following-sibling` should be empty on an attribute nodes. Instead, an array index out of bounds exception is thrown:

```xquery
let $d := <a b="c"/>
return empty($d/@b/preceding-sibling::*)
```
