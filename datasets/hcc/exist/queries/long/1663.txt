### What did you expect

for the query below: no NPE, this XML file:

```xml
<root attr="1">
  <child/>
</root>
```

### Describe how to reproduce or add a test

To reproduce, run these lines of code:

```xquery
let $f  := xmldb:store('/db/apps', 'ttt.xml', <root attr="1"/>)
let $u  := update insert <child/> into doc($f)/root

return doc($f)
```

If you remove `attr=“1”` from the first node, the "update insert” statement will be successful.
If you add a `<!— comment —>` under the root node, the "update insert” statement will be successful.
