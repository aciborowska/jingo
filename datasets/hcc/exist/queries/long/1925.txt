On [http://exist-db.org/exist/apps/doc/xqsuite#D3.19](http://exist-db.org/exist/apps/doc/xqsuite#D3.19) the provided example seems to be outdated in the code for `suite.xql`.

The query:

```xquery
import module namespace test="http://exist-db.org/xquery/xqsuite" at "resource:org/exist/xquery/lib/xqsuite/xqsuite.xql";

test:suite( inspect:module-functions(xs:anyURI("math.xql")) )
````

has 0 results when tested in exist-db 4.2.0 and 4.1.0, while [Joe's approach](https://gist.github.com/joewiz/fa32be80749a69fcb8da) with `util:list-functions` instead of `inspect:module-functions` yields the expected testsuite outcome.

*   exist-db version: `4.2.0`
*   documentation version: current website
