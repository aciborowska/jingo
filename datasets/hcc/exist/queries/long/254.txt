The following script:

declare namespace sql="http://exist-db.org/xquery/sql";
let $conn := sql:get-connection("com.mysql.jdbc.Driver", "jdbc:mysql://localhost/codif", "root", "")
let $stmt := sql:prepare($conn, "insert into table1 (foo) values (?)")
let $params := sql:parameters<sql:param sql:type="varchar">/sql:param/sql:parameters
return sql:execute($conn, $stmt, $params, false())

causes an exception.

I traced the problem to method setParametersOnPreparedStatement() in ExecuteFunction.java in the sql module.  At line 433, variable child is set:

  Node child = param.getFirstChild();

It is obviously set to null if the <param> element is empty.  Then at line 438 it is used:

  final String  value = child.getNodeValue();

which crashes on a NullPointerException.
