When sm:is-authenticated() is called in a module, the function throws an error.

test.xqm:

``` xquery
xquery version "3.0";

module namespace test="http://exist-db.org/test";

declare namespace sm="http://exist-db.org/xquery/securitymanager";

declare function test:is-authenticated() as xs:boolean {
    sm:is-authenticated()
};
```

test.xql:

``` xquery
xquery version "3.0";

import module namespace test="http://exist-db.org/test" at "test.xqm";

test:is-authenticated()
```
