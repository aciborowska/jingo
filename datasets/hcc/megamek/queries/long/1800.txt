### Environment
MekHQ 0.47.4 and below.
Windows
Java version 1.8.0_241

### Description
Units using Jump Jets to land on bridges over terrains that use Bog Down rules end up Stuck in the terrain below the bridge. I've tested normal walk/run, jumping and attempts to move using Climb Mode following the suggestions provided on the Facebook group. The results seems to work normally for ground movement but can be anywhere from mildly annoying to flat out disastrous for units that Jump onto the bridge. Screenshots to follow.
