### Description
If a targeted unit has a long name  like "Jump Support Infantry Star Corps Jump Assault Infantry (Capellan Confederation (Local Forces))," the text in the Unit Display weapon tab is shifted left so that the standard text (Name, Target, Range, etc.) displays the first letter and an ellipsis.
