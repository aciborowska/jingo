Added a function count() to count elements in a group and return a two tuple with original group value and count.

Allows to write code like:
```
val words = input flatMap { _.toLowerCase().split("""\W+""") filter 
{ _ != "" }
 }
val counts = words groupBy 
{ x => x }
 count()
```
