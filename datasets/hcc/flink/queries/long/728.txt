Instead of
```
Plan p = env.CreateProgramPlan();
p.registerCachedFile(path, "name");
PlanExecutor l = new Local-/RemoteExecutor();
l.executePlan(p);
```
you can now write
```
env.registerCachedFile(path, "name");
env.execute();
```
