Oracle insert statement exists encrypt bug for 4.0.0 release version. Because of oracle insert statement definition modification, the columnNames couldn't be extracted. The bug will be fixed in 4.0.1 version. `InsertValuesExtractor` could extract column segements by `InsertValuesClause` rule, but oracle insert statement `insertValuesClause` lack of `columnNames` , which cause encrypt problem for extracting column segements.
