## Bug Report

### Steps to Reproduce:

- Create profile with two rules e.g. : 
```
{
    "schemaVersion": "0.1",
    "description": "Testing Profile",
    "fields": [
        {
            "name": "foo"
        }
    ],
    "rules": [
        {
            "rule": "rule1",
            "constraints": [
                {
                    "field": "foo",
                    "is": "equalTo",
                    "value": "goodbye"
                }
            ]
        },
        {
            "rule": "rule2",
            "constraints": [
                {
                        "field": "foo",
                        "is": "ofType",
                        "value": "string"
                }
            ]

}
```
