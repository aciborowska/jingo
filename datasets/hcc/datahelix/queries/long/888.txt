## Bug Report

### Steps to Reproduce:
run a simple profile with multiple fields with the cartesian walker
e.g. 
field a in set [1, 2, 3]
field b in set [1, 2, 3]
field c in set [1, 2, 3]

### Expected Result:
all fields randomly generate

### Actual Result:
only 1 field generates randomly, the others generate the same
