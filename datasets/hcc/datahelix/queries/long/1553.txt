## Bug Report
if constraint without then does not give helpful error message 

### Steps to Reproduce:

Run the profile
```
{
  "fields": [
      {
          "name": "morning",
          "type": "time",
          "nullable": false
      }
  ],
          "constraints": [
              {
                  "if": {
                      "field": "morning",
                      "before": "12:00:00"
                  }
              }
          ]
}

```


### Expected Result:
A messege indicating that an if statement is malformed, potentially giving line numbers.

### Actual Result:
```
* Constraint must not be null
```
