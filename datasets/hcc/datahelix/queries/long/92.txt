## Bug Report

### Steps to Reproduce:
```javascript
{
	"schemaVersion": "v3",
	"fields": [
		{ "name": "title" }
	],
	"rules": [
		{ "field": "title", "is": "inSet", "values": [ "mr", "mrs" ] },
		{ "not": { "field": "title", "is": "inSet", "values": [ "mrs", "dr" ] } }
	]
}

```
### Expected Result:
Row where `title` = `mr`.

### Actual Result:
Only null row.
