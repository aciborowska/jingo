## Bug Report

### Steps to Reproduce:
 1. Create field foo
 2. Add constraint is longerThan with large bigDecimal value (e.g. 1e18)
 3. Generate

### Expected Result:
~Generation of strings of length 1e18 characters or above~
An error message should be presented for any attempt to create a string above the maximum amount allowed in Java.

Strings in Java can only be created up-to Integer.MAX_VALUE in length; therefore the largest possible strings would be 2147483647 long.
