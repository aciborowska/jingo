## Bug Report

### Steps to Reproduce:
```
{
    "schemaVersion": "0.1",
    "fields": [
        {
            "name": "shortString"
        }
    ],
    "rules": [
        {
            "constraints": [
                {
                    "field": "shortString",
                    "is": "ofType",
                    "value": "string"
                },
                {
                    "field": "shortString",
                    "is": "shorterThan",
                    "value": 2
                },
                {
                    "not" : {
                        "field": "shortString",
                        "is": "null"
                    }
                }
            ]
        }
    ]
}
```
### Expected Result:
The strings generated should consist of only one character

### Actual Result:
Strings upto length three are generated

### Additional Context:
It appears that for any shorter than value, the actual limit is equal to 3x higher than the one specified
