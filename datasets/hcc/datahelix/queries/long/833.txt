## Bug Report

### Steps to Reproduce:
1. Create a field foo
2. foo is ofType String
3. foo is not null
4. Generate data in random mode (current default) with a max row limit 

### Expected Result:
Random strings generated.

| constraint | expected output |
| --- | --- |
| - | random strings of length 0..255 |

### Actual Result:
"Lorem Ipsum" repeated for the max row limit. This does not appear to be random to the user.
