## Bug Report
Maximum value for longerThan is 1e20. This is much higher than actually possible.

### Steps to Reproduce:
 1. Create field foo
 2. Add constraint is longerThan with large value (e.g. 1e4)
 3. Generate

### Expected Result:
Generation of strings of length 1e4 characters or above/InvalidProfileException due to value being out of bounds

### Actual Result:
Stack overflow exception at:
```
com.scottlogic.deg.generator.generation.RegexStringGenerator.generateRandomStringInternal(RegexStringGenerator.java:284)
```

### Sample Profile:
```
{
    "schemaVersion": "0.1",
    "description": "Testing Profile",
    "fields": [
        {
            "name": "foo"
        }
    ],
    "rules": [
        {
            "rule": "rule1",
            "constraints": [
                {
                    "field": "foo",
                    "is": "longerThan",
                    "value": 1e4
                }
            ]
        }
    ]
}
```
