## Bug Report

In generate mode, running the generator without an `--output-path` option redirects output to standard output.

In violate mode, it throws an exception

### Steps to Reproduce:
 1. Run a command such as `java -jar generator.jar violate --profile-file=[profile]` with no `--output-path=...` option specified.

### Expected Result:
A human-readable error message explaining that the `--output-path` option is compulsary in violate mode.
