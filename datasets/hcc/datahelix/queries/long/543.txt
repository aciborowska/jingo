## Bug Report

### Steps to Reproduce:
 1. Update `BaseModule` to _bind_ the `IterationVisualiser` to `ReductiveIterationVisualiser`
 2. Run the generator with the `-w=reductive` switch

### Expected Result:
A `reductive-walker` directory should be created in the working directory of the generator. For every reductive-iteration a tree visualisation should be created.

### Actual Result:
Google Guice throws a provisioning error because the required dependencies of `ReductiveIterationVisualiser` cannot be satisfied.
