
### Steps to Reproduce

1.  Open example project (see below), start micronaut `./mvnw compile mn:run`
2. Open terminal where curl command line tool available;
    Run `curl -v http://localhost:8080/test -o output.jpg`

### Expected Behaviour

Downloaded file `output.jpg` should contain 60,414 bytes.

### Actual Behaviour

Ouput file `output.jpg` contains 0 bytes. Response has no header `Content-Length`.


### Environment Information

- **Operating System**: Windows
- **Micronaut Version:** 2.0.0-M4
- **JDK Version:** JDK11, JDK14

### Example Application

https://github.com/iovuio/ll-mn2-byte-array-response-body-broken

### Note

Title says broken in 2.0.0 because the exact same scenario run on micronaut 1.3.4 works as expected.

This issue is linked to #3048. Please allow passing the input stream directly as the response body argument.
