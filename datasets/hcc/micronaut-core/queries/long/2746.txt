### Steps to Reproduce

``` 
       MediaType mt =
                new MediaType("application/json",
                        Map.of(MediaType.CHARSET_PARAMETER, StandardCharsets.UTF_8.name()));
```

### Expected Behaviour

variable mt will contain an instance of media type, which would contain parameter "charset" with value "UTF-8"

### Actual Behaviour

```
Exception in thread "main" java.lang.UnsupportedOperationException
	at java.base/java.util.AbstractMap.put(AbstractMap.java:209)
	at java.base/java.util.AbstractMap.putAll(AbstractMap.java:281)
	at io.micronaut.http.MediaType.<init>(MediaType.java:442)
	at io.micronaut.http.MediaType.<init>(MediaType.java:382)
	at xxx(xxx.java:182)
```

(the problem is that the constructors creates immutable map to which it tries to add passed parameters, see 

(create) https://github.com/micronaut-projects/micronaut-core/blob/1.3.x/http/src/main/java/io/micronaut/http/MediaType.java#L419

(attempted putAll) https://github.com/micronaut-projects/micronaut-core/blob/1.3.x/http/src/main/java/io/micronaut/http/MediaType.java#L445

### Workaround available

```
        MediaType mt =
                new MediaType("application/json;" + MediaType.CHARSET_PARAMETER + "=" + StandardCharsets.UTF_8.name());

        System.out.println(mt.getCharset().get());
```

### Environment Information

- **Operating System**: N/A
- **Micronaut Version:** 1.3.0
- **JDK Version:** 11.0.6
