### Steps to Reproduce

1. Create websocket server e.g. `@ServerWebSocket("/ws/chat")`
2. Try to connect using url that contains trailing slash (before params) e.g. `/ws/chat/?token=x`

### Expected Behaviour

WebSocket should be matched regardless trailing slash and connect properly.

### Actual Behaviour

WebSocket is not found, 404 is being returned.

### Environment Information

- **Operating System**: MacOS
- **Micronaut Version:** 2.0.0
- **JDK Version:** 1.8
