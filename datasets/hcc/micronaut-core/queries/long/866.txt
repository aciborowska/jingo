In Micronaut:

```
    @Status(HttpStatus.CREATED)
    @Get(value = "/voidreturn")
    public String voidReturn() {
         return "foo";
    }
```

returns 201.

But with `void`


```
    @Status(HttpStatus.CREATED)
    @Get(value = "/voidreturn")
    public void voidReturn() {
    }
```

returns 200.

In SpringBoot the following code returns 201.

```
 @ResponseStatus(HttpStatus.CREATED)
    @GetMapping
    public void voidReturn() {

    }
}
```

It probably make sense to exhibit the same behaviour wether a return type is specified or not.
