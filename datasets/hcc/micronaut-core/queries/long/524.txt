Steps to reproduce: 

Create app

`mn create-app example.micronaut`

Create interface: 

`src/main/java/example/VatValidator .java`
```
package example;

public interface VatValidator {

    String hello();
}
```

Create bean:

`src/main/java/example/VatClient .java`
```
package example;

import javax.inject.Singleton;

@Singleton
public class VatClient implements VatValidator {
    @Override
    public String hello() {
        return null;
    }
}
```

Create bean in the test classpath to replace previous bean

`src/test/java/example/MockVatValidator .java`
```
package example;

import io.micronaut.context.annotation.Replaces;
import io.micronaut.context.annotation.Requires;
import io.micronaut.context.env.Environment;

import javax.inject.Singleton;

@Requires(env = Environment.TEST)
@Replaces(VatClient.class)
@Singleton
public class MockVatValidator implements VatValidator {
    @Override
    public String hello() {
        return null;
    }
}
```

Create a test which verfies `MockVatValidator` is used: 

`src/test/java/example/VatValidatorTest .java`
```
package example;

import io.micronaut.context.ApplicationContext;
import io.micronaut.context.env.Environment;
import org.junit.Test;

import static org.junit.Assert.assertTrue;

public class VatValidatorTest {

    @Test
    public void testFunctionClientReplacement() {
        ApplicationContext context = ApplicationContext.run(Environment.TEST);

        VatValidator bean = context.getBean(VatValidator.class);
        assertTrue(bean instanceof MockVatValidator);

        context.close();
        context.stop();
    }
}
``` 

The previous test passes. 

Now, annotate `VatClient` with `@FunctionClient ` instead of `@Singleton`. The test fails: 

```
package example;

import io.micronaut.function.client.FunctionClient;

@FunctionClient
public class VatClient implements VatValidator {
    @Override
    public String hello() {
        return null;
    }
}
```
