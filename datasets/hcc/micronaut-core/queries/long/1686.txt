### Steps to Reproduce

`mn create-app size-ko`

```java
@Controller("/")
public class SizeController {
    @Put(value = "/", consumes = MediaType.APPLICATION_OCTET_STREAM, produces = MediaType.TEXT_PLAIN)
    HttpResponse<String> put(@Size(max = 1024) @Body byte[] content)  {
        return HttpResponse.created("hello");
    }
}
```

Then
`./gradlew run`


### Expected Behaviour

Build is OK and application start

`@Size` javadoc says Array element is allowed.

> <li>Array (array length is evaluated)</li>

Was OK in version 1.0.4
