### Steps to Reproduce

- Create app `mn create-app example.micronaut.app`

Create a Controller with a POJO in an endpoint method: 

```
package example.micronaut;

import io.micronaut.http.HttpStatus;
import io.micronaut.http.annotation.Body;
import io.micronaut.http.annotation.Controller;
import io.micronaut.http.annotation.Post;
import io.micronaut.http.annotation.Status;

@Controller
public class HomeController {

    @Post
    @Status(HttpStatus.OK)
    public void index(@Body SavePuzzleChallenge dto) {

    }
}
```

POJO is annotated with `@Introspected` and contains a field of type `LocalDateTime`

```
package example.micronaut;

import io.micronaut.core.annotation.Introspected;

import javax.annotation.Nonnull;
import javax.validation.constraints.NotNull;
import java.time.LocalDateTime;

@Introspected
public class SavePuzzleChallenge {

    @Nonnull
    @NotNull
    private LocalDateTime dateSent;

    public SavePuzzleChallenge() {

    }
    public SavePuzzleChallenge(LocalDateTime dateSent) {
        this.dateSent = dateSent;
    }


    public LocalDateTime getDateSent() {
        return dateSent;
    }

    public void setDateSent(LocalDateTime dateSent) {
        this.dateSent = dateSent;
    }
}
```

Create a test:

```
package example.micronaut

import io.micronaut.context.ApplicationContext
import io.micronaut.http.HttpRequest
import io.micronaut.http.client.BlockingHttpClient
import io.micronaut.http.client.HttpClient
import io.micronaut.runtime.server.EmbeddedServer
import spock.lang.AutoCleanup
import spock.lang.Shared
import spock.lang.Specification

import java.time.LocalDateTime

class HomeControllerSpec extends Specification {

    @AutoCleanup
    @Shared
    EmbeddedServer embeddedServer = ApplicationContext.run(EmbeddedServer)

    @AutoCleanup
    @Shared
    HttpClient httpClient = embeddedServer.applicationContext.createBean(HttpClient, embeddedServer.URL)

    BlockingHttpClient getClient() {
        httpClient.toBlocking()
    }

    void "can post with a POJO with a LocalDateTime property"() {
        when:
        client.exchange(HttpRequest.POST("/", new SavePuzzleChallenge(LocalDateTime.now())))

        then:
        noExceptionThrown()
    }
}
```

If you set `jackson.module-scan: true` test pass. 

if you set it to false: 

`src/main/resources/application.yml`

```
micronaut:
    application:
        name: app
jackson:
    bean-introspection-module: true
    module-scan: true
```

and set http-client logger to trace:

```
    <logger name="io.micronaut.http.client" level="TRACE"/>
```
