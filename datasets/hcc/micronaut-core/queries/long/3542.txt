Micronaut 2.0.0.RC1 with Jaxrs and HeaderParam throws: `Error instantiating bean of type [io.micronaut.web.router.DefaultRouter]`. This happens when you have `@HeaderParam` as controller method argument.

### Task List

- [x] Steps to reproduce provided
- [x] Stacktrace (if present) provided
- [x] Example that reproduces the problem uploaded to Github
- [x] Full description of the issue provided (see below)

### Steps to Reproduce

1. Create micronaut app 2.0.0.RC1 with Jaxrs
2. Create controller with `@HeaderParam`
3. Create test and run it

### Expected Behaviour

Header is populated, app works normally.

### Actual Behaviour

App crashes

### Environment Information

- **Operating System**: MacOS
- **Micronaut Version:** 2.0.0.RC1
- **JDK Version:** Java8

### Example Application

- https://github.com/asodja/micronaut-jaxrs-reproducer
