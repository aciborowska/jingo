### Steps to Reproduce

1. Clone [this repo](https://github.com/arudge/micronaut-issue-croninator)
2. Run the following test -> https://github.com/arudge/micronaut-issue-croninator/blob/master/src/test/groovy/croninator/FailCronSpec.groovy
3. See exception thrown in test output (java.lang.IllegalArgumentException)
4. Alternative option would be to edit the cron expression to only have 5 values in [ScheduledFixedRateSpec](https://github.com/micronaut-projects/micronaut-core/blob/master/runtime/src/test/groovy/io/micronaut/scheduling/ScheduledFixedRateSpec.groovy)  

### Expected Behaviour

[As documented](https://docs.micronaut.io/latest/api/io/micronaut/scheduling/cron/CronExpression.html), you should be able omit the seconds cron entry.

### Actual Behaviour

Micronaut server does not run scheduled job (server still starts though) and provides the following log:
