### Steps to Reproduce

1. Make a request with the RxHttpClient that has a deserialized type of nested lists such as `<List<List<Event>>` using Argument.listOf as shown below:

```
Flowable<HttpResponse<List<List<Event>>>> call = client.exchange(
                POST(QUERY_ENDPOINT, request).contentType(MediaType.APPLICATION_JSON),
                Argument.listOf(Argument.listOf(Event.class).getType())
        );
```

### Expected Behaviour

Expected behavior is to get a `List<List<Event>>`

### Actual Behaviour

Actual behavior is getting a `List<List<LinkedHashMap>>` without any errors being indicated.
