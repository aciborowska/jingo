### Steps to Reproduce

1. Create the following controller in a micronaut `1.3.4` and `2.0.0.M2` application:
```
@Controller
class Controller {
	@Get("/test1")
	fun test1(): Single<HttpResponse<Flowable<ByteArray>>> {
		return Single.just(HttpResponse.ok(Flowable.empty()))
	}
	@Get("/test2")
	fun test2(): Single<HttpResponse<Flowable<ByteArray>>> {
		return Single.just(HttpResponse.ok(Flowable.just("".toByteArray())))
	}
}
```
2. Curl both endpoints in both applications

### Expected Behaviour

Both applications' endpoints should return successfully

### Actual Behaviour

Endpoints return successfully in `1.3.4`, and fail in `2.0.0.M2`.
