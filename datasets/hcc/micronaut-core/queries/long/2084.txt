### Steps to Reproduce

1. Enable Sessions on the `hello-world-java` example project
2. Create `SessionCreatedEvent` and `SessionExpiredEvent` listeners
3. Add Session parameter to the Controller method
4. Send a request

### Expected Behaviour
Sessions should expire after an interval

### Actual Behaviour
Sessions don't expire, `SessionExpiredEvent`s are never fired.
Setting `micronaut.session.max-inactive-interval=1m` with any value doesn't seem to work either.

### Environment Information

- **Operating System**: Ubuntu 
- **Micronaut Version:** 1.2.0
- **JDK Version:** 1.8
