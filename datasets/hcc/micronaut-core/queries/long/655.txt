### Steps to Reproduce

Create an App using the CLI with the feature `hibernate-jpa` and build `maven`:
`mn create-app hibernate-jpa-app-vanilla --features hibernate-jpa --build maven`

Build the app:
`mvn clean package`

### Expected Behaviour

Build should be successful

### Actual Behaviour

Build fails with the following stacktrace:
```
ERROR] Failed to execute goal org.apache.maven.plugins:maven-compiler-plugin:3.7.0:compile (default-compile) on project hibernate-jpa-app-vanilla: Resolution of annotationProcessorPath dependencies failed: Missing:
[ERROR] ----------
[ERROR] 1) javax.persistence:javax.persistence-api:jar:1.0.0.RC1
[ERROR] 
[ERROR] Try downloading the file manually from the project website.
```

When looking at the pom.xml, the problem seems to be in the `maven-compiler-plugin` configuration when it refers to the `persistance-api` dependency:
```
<plugin>
                    <groupId>org.apache.maven.plugins</groupId>
                    <artifactId>maven-compiler-plugin</artifactId>
                    <version>3.7.0</version>
                    <configuration>
                        <source>${jdk.version}</source>
                        <target>${jdk.version}</target>
                        <compilerArgs>
                            <arg>-parameters</arg>
                        </compilerArgs>
                        <annotationProcessorPaths>
                            <path>
                                <groupId>javax.persistence</groupId>
                                <artifactId>javax.persistence-api</artifactId>
                                <version>${micronaut.version}</version>      <-- HERE BE DRAGONS
                            </path>
                            ...
                        </annotationProcessorPaths>
                    </configuration>
 </plugin>
```

### Environment Information

- **Operating System**: Mac OS Mojave
- **Micronaut Version:** 1.0.0.RC1
- **JDK Version:** 10,11
