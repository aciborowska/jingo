### Steps to Reproduce

1. Declarative Http Client with @Retryable
2. An endpoint with POST that require non empty body
3. Send request using the client towards the endpoint which returns >399 response

### Expected Behaviour

Client retries due to non successful response, all requests contains same body.

### Actual Behaviour

First attempt body is correct, following attempts the body is:

`    "body" : {
      "contiguous" : true,
      "direct" : false,
      "readable" : false,
      "writable" : false,
      "readOnly" : false
    }`

### Environment Information

- **Operating System**: MacOsx
- **Micronaut Version:** 2.0.0
- **JDK Version:** 11
