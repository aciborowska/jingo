### Steps to Reproduce

1. Have following code on the classpath:

```
import java.util.*
import javax.persistence.*

@Entity
class Event : Item<Event>()

@MappedSuperclass
abstract class Item<T : Item<T>> {

    var id: Long? = null

    var revisions: MutableList<T> = ArrayList()
}
```

2. Run gradle build in Micronaut project.
3. You will get this exception:

https://gist.github.com/knyttl/4b34326939184910e93e7e4b3c03012b

### Expected Behaviour

1. The exception should be more informative to be able to find out what causes the problem.
2. Gradle build should not fail.

### Actual Behaviour

1. We get StackOverflow error with not enough information.
2. Gradle build fails.

### Environment Information

- **Operating System**: Any
- **Micronaut Version:** 1.1.0
- **JDK Version:** JDK 11
