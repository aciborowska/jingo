I was creating an example using a Client created at `test` package and when trying to perform a delete operation a message is shown telling that the operation is not allowed.

### Task List

- [Y] Steps to reproduce provided

### Steps to Reproduce

1. Clone the project 
2. Checkout the branch `issue-client-at-test-dir`
3. Execute the command `gradle test`

### Expected Behavior

All the test pass as successful 

### Actual Behavior

The `delete` operation throws an exception

### Workaround 

There are 2 ways to solve this problem actually

- 1.- Move the client to `main` sources

```
src/main
├── groovy
   └── com
       └── nearsoft
            └── micronaut
                  ├── client
                  └──PersonClient.groovy

```

- 2.- Override the method on `PersonClient` interface

```
@Client("/people")
interface PersonClient extends PersonApi {

    @Delete("/{id}")
    HttpResponse delete(Long id)

}
```
