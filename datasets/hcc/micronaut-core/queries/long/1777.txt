### Steps to Reproduce

1. create a project with graal-native-image feature
2. create a class (with any name), and a method like 
`
@EventListener
    void onStartup(ServiceStartedEvent event) {
        System.out.println("yes");
    }
`
3. do assemble and run docker-build.sh
4. run the compiled image

Micronaut should be startable
