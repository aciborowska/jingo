### Steps to Reproduce

1. Create controller with the following signature:
```java
@Controller("/broken-controller")
public class BrokenController
{
    @Get("/list{?full}")
    public String getFull(@QueryValue(value = "full", defaultValue = "true")
                          @Nullable Boolean full)
    {
        return "full";
    }

    @Get("/{id}")
    public String getById(@PathVariable("id") Integer id)
    {
        return "byId";
    }
}
```
2. Invoke with path `/broken-controller/list?full=false`

### Expected Behaviour

`BrokenController#getFull` is invoked

### Actual Behaviour

```json
{
  "message": "More than 1 route matched the incoming request. The following routes matched /broken-controller/list: GET - /broken-controller/list, GET - /broken-controller/list",
  "_links": {
    "self": {
      "href": "/broken-controller/list?full=false",
      "templated": false
    }
  }
}
```

### Environment Information

- **Micronaut Version:** 1.2.1
