### Steps to Reproduce

1. Set `micronaut.session.http.cookie-max-age: 60` in `application.yml`
2. Run `./gradlew run`
3. Navigate to `http://localhost:8080`

### Expected Behaviour

The server starts and a session cookie with a maximum age of 60 seconds is configured (doesn't have to be sent to the client).
