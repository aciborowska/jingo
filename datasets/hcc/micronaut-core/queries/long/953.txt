### Steps to Reproduce

1. `mkdir micronaut-issuer-issue`
2. `cd micronaut-issuer-issue`
3. `mn create-app --inplace`
4. Create a file: `src/main/java/micronaut/issuer/FooConfigurationProperties.java`

```
package micronaut.issuer;

import io.micronaut.context.annotation.ConfigurationProperties;

@ConfigurationProperties("foo")
public class FooConfigurationProperties {

    private String issuer;

    public String getIssuer() {
        return issuer;
    }

    public void setIssuer(String issuer) {
        this.issuer = issuer;
    }
}
```

`./gradlew compileJava`

It should compile.
