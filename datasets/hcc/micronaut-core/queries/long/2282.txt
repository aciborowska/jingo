NPE error while trying to deserialize string "null".
In this case `ObjectMapper` returns null value, but `JacksonObjectSerializer` use Optional.of method to wrap result.
And it throws a `NullPointerException` then.

### Steps to Reproduce

1. Inject or create JaksonObjectSerialize
2. execute deserialize method with string "null"
```java
    ObjectMapper objectMapper = new ObjectMapperFactory().objectMapper(null, null);
    new JacksonObjectSerializer(objectMapper).deserialize("null".getBytes());
```
### Expected Behaviour

Method should return empty Optional

### Actual Behaviour

NullPointerException is thrown

```stacktrace
java.lang.NullPointerException
	at java.util.Objects.requireNonNull(Objects.java:203)
	at java.util.Optional.<init>(Optional.java:96)
	at java.util.Optional.of(Optional.java:108)
	at io.micronaut.jackson.serialize.JacksonObjectSerializer.deserialize(JacksonObjectSerializer.java:68)
	at io.micronaut.core.serialize.ObjectSerializer.deserialize(ObjectSerializer.java:111)
```

### Environment Information

- **Operating System**: Windows 10 Pro
- **Micronaut Version:** 1.2.5
- **JDK Version:** 1.8
