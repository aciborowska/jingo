### Description
If you key in a single character feature name to `create-app` command, an error is produced.

### Steps to Reproduce

1. Open Micronaut CLI `mn`
2. Type the command `create-app com.nareshak.demo.sample-service --lang groovy --features f`
3. Press enter to execute the command

### Expected Behaviour

The CLI should print 'invalid feature' warning as 'f' is not a valid feature.

### Actual Behaviour

The following error is produced
```
| Error Command [mn] error: String index out of range: 2 (Use --stacktrace to see the full trace)
```

### Environment Information

- **Operating System**: macOS
- **Micronaut Version:** 1.1.0
- **JDK Version:** 1.8.0_202
