### Steps to Reproduce

1. Create controller with a `@Post` method, taking `HttpRequest` as an argument.
   ```java
   @Post("/test")
    public HttpResponse<?> test(HttpRequest<ObjectNode> request)
    {
        LOG.info(request.getBody().get().toString());
        return HttpResponse.ok();
    }
   ```
2. Invoke it with any json body

### Expected Behaviour

Body should be available via `request.getBody()`

### Actual Behaviour

Body is not available

### Environment Information

- **Micronaut Version:** 1.1.0.RC1
