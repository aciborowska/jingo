When Gradle incremental compilation is enabled the compilation intermittently fails with errors like:

```
error: Unexpected error: Attempt to recreate a file for type io.micronaut.oci.objectstorage.$ObjectStorageClientTestDefinition
error: Unexpected error: Attempt to recreate a file for type io.micronaut.oci.objectstorage.$SdkTestDefinition
```

Unclear whether this is a Gradle bug or Micronaut bug at this stage.

### Steps to Reproduce

1. Checking git@github.com:micronaut-projects/micronaut-oracle-cloud.git
2. Make a change to any of the classes and rerun compilation
3. After a few changes the error will appear


### Environment Information

- **Operating System**: Any
- **Micronaut Version:** 2.0.0
- **JDK Version:** Any
