Given:

```java
public interface H {
    String test(String name);
}
```

And implementation:

```java
import io.micronaut.aop.simple.Mutating;

@Mutating("name")
public class HImpl implements H {

    @Override
    public String test(String name) {
        return "default " + name;
    }
}
```

If a replacement is defined via factory:

```java
@Factory
public class HFactory {

    @Singleton
    @Replaces(HImpl.class)
    H myInterface() {
        return name -> "replacement " + name;
    }
}
```

The bean is not replaced.  This is causing https://github.com/micronaut-projects/micronaut-test/issues/39
