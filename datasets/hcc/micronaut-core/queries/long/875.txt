### Steps to Reproduce

1. Create new micronaut application with management endpoints:
```
mn create-app micronaut-logger-endpoint --features management
```
2. Run the application
```
./gradlew run
```
3. Try to reset the loglevel of a specific logger over http and supply null as configuredLevel.
```
curl -XPOST http://localhost:8080/loggers/io.micronaut -H "Content-Type: application/json" -d '{"configuredLevel":null}'
``` 

### Expected Behaviour

The loglevel should be reset like in the test `io.micronaut.management.endpoint.loggers.LoggersEndpointSpec#test that log levels on logger "#name" can be configured via the loggers endpoint`

### Actual Behaviour
