### Steps to Reproduce

Doing something like:

```java
 @Bean
    GreeterGrpc.GreeterBlockingStub blockingStub() {
        return GreeterGrpc.newBlockingStub(
                channel
        );
    }
```

Where `GreeterGrpc.GreeterBlockingStub` is a final class leads to a VerifierError:

```
used by: java.lang.VerifyError: Cannot inherit from final class
	at java.base/java.lang.ClassLoader.defineClass1(Native Method)
	at java.base/java.lang.ClassLoader.defineClass(ClassLoader.java:1016)
	at java.base/java.security.SecureClassLoader.defineClass(SecureClassLoader.java:174)
	
```

### Expected Behaviour

Compilation should fail.

### Actual Behaviour

Compilation succeeds and the error is reported at runtime.

### Environment Information

- **Micronaut Version:** 1.0.2
