### Steps to Reproduce

1. Create a topic listener with a key parameter of type UUID annotated with @KafkaKey.
2. Start the application with the listener.
3. Submit to the listened topic message with the key is not converted to UUID.
4. Check the logs.

### Expected Behaviour

Get conversion exception in UUID.

### Actual Behaviour

The listener is disconnected from the group in which it is the only member. No information is provided about the incident.

### Environment Information

- **Operating System**: windows
- **Micronaut Version:** 1.0.1
- **JDK Version:** 1.8
