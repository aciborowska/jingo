### Steps to Reproduce

Define method with `defaultValue`:

```java
@Get("/{?number,size}")
    List<NameDTO> all(@PathVariable(value = "number", defaultValue = "0") int number, @PathVariable(value = "size", defaultValue = "10") int size) 
```

### Expected Behaviour

Should compile

### Actual Behaviour

Fails compilation with:

```
 The uri variable [number] is optional, but the corresponding method argument [int number] is not defined as an Optional or annotated with the javax.annotation.Nullable annotation.
```
