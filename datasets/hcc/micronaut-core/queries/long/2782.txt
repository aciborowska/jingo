### Steps to Reproduce

Create a controller method with a single letter path. 
Backslash at end of parent path defined on controller will be removed. Doesn't happen if path is more than one letter.

### Example

```kotlin
import io.micronaut.http.annotation.Controller
import io.micronaut.http.annotation.Get
import javax.annotation.security.PermitAll

@PermitAll
@Controller("/demo/")
class DemoController {

    @Get("a") // Maps incorrectly to /demoa 
    fun a() = "a"

    @Get("ab") // Maps to /demo/ab
    fun ab() = "ab"
}

```

Log:

```
17:36:58.689 [main] DEBUG i.m.web.router.DefaultRouteBuilder - Created Route: GET /demoa -> DemoController#String a() (application/json )
17:36:58.689 [main] DEBUG i.m.web.router.DefaultRouteBuilder - Created Route: HEAD /demoa -> DemoController#String a() (application/json )
17:36:58.690 [main] DEBUG i.m.web.router.DefaultRouteBuilder - Created Route: GET /demo/ab -> DemoController#String ab() (application/json )
17:36:58.690 [main] DEBUG i.m.web.router.DefaultRouteBuilder - Created Route: HEAD /demo/ab -> DemoController#String ab() (application/json )
```

### Expected Behaviour

Controller method should map to /demo/a instead of /demoa

### Environment Information

- **Operating System**:  macOS
- **Micronaut Version:** 1.2.10, 1.3.0
- **JDK Version:** 11
