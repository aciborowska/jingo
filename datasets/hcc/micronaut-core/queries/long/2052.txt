### Steps to Reproduce

1. Using CLI create new app (Groovy or Java) with Spock as testing framework 
2. Using CLI create controller - MyTestController
3. Open MyTestControllerSpec.groovy
4. Create 2nd test method, e.g. simply copy **void "test index"()**  to **void "test index 2"()**
5. Execute ./gradlew build

### Expected Behaviour

Expecting tests to pass
