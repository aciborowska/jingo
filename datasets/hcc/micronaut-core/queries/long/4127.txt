### Steps to Reproduce

1. Create new micronaut kotlin project with http client
2. Create new controller(Ex: FooController)
3. Annotate controller class with `@Validated`
4. Build

### Expected Behaviour

Should create $Intercepted class for controller annotated with one `@Generated` at `target.classes.*some_package*` after build

### Actual Behaviour

Duplicated `@Generated` annotation on controller intercepted class(Ex:`$FooControllerDefinition$Intercepted.class`) after build.

- **Operating System**: Catalina
- **Micronaut Version:** 2.0.2
- **JDK Version:** 11
