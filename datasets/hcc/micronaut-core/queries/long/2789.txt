I just started using Micronaut with a completely fresh install, however, when i try to use create-function using `mn`, it falls over when running up the application.

### Steps to Reproduce
on https://docs.micronaut.io/latest/guide/index.html#serverlessFunctions
```
$ mn create-function hello-world
$ cd hello-world
$ ./gradlew assemble
$ echo '{"name":"hello"}' | java -jar build/libs/hello-world-0.1-all.jar
```

### Expected Behaviour

Result in the following output being printed to standard out:

```
{"name":"hello"}
```
