### Steps to Reproduce
- mn create-app test --features groovy
- cd test
- mn create-controller org.test.Test
- in build.gradle add deps
```
    compile "io.micronaut:micronaut-views"
    compile "org.thymeleaf:thymeleaf:3.0.9.RELEASE"
```
- add src/main/resources/views/test.html
```
<!DOCTYPE html>
<html th:fragment="layout (title)" xmlns:th="http://www.thymeleaf.org">
<head>
    <title>test</title>
</head>
<body th:text="${message}">

</body>
</html>
```
- in org.test.TestController
```
package org.test

import io.micronaut.http.HttpResponse
import io.micronaut.http.annotation.Controller
import io.micronaut.http.annotation.Get
import io.micronaut.views.View


@Controller("/test")
class TestController {

    @Get
    @View('test')
    HttpResponse index() {
        HttpResponse.ok(['message':'test'])
    }
}
```
- gradle assemble
- java -jar build\libs\test-0.1-all.jar

### Expected Behaviour

http://localhost:8080/test browser renders test

### Actual Behaviour

http://localhost:8080/test returns a 404

### Note
The page renders as expected when running application not as a jar using gradle.

### Environment Information

- Windows 10 64 bit
- 1.0.0.RC3
- jdk 1.8.0_181
