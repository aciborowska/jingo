### Steps to Reproduce

1. Start Eureka and two services with digits in their names (gr8crm-tag-service & gr8crm-notification-service)
2. Make sure you DEBUG log io.micronaut.http.client
3. See how incorrect url is used to contact eureka

### Expected Behaviour

Sending HTTP Request: GET /eureka/apps/gr8crm-tag-service

### Actual Behaviour

 Sending HTTP Request: GET /eureka/apps/gr8-crm-tag-service
Response is 404
