### Steps to Reproduce

1. mn create-app micro --features hibernate-gorm,groovy,spock
2. cd micro
3. create gorm entity in micro package
```
package micro

import grails.gorm.annotation.Entity
import org.grails.datastore.gorm.GormEntity

@Entity
class Person implements GormEntity<Person> {
    String firstName
    String lastName
}
```
4. gradlew run

### Expected Behaviour

Application should start on localhost

### Actual Behaviour
