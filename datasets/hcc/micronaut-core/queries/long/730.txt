Application still try to secure endpoints, also even if they are marked with anonymous access.
### Steps to Reproduce
Clone or download: https://github.com/micronaut-guides/micronaut-security-jwt-groovy
Edit application.yml change configuration from true to false for :  micronaut.security.enabled=false
### Expected Behaviour
Should disable security validations for all endpoints

@Controller("/") // <2>
class HomeController {

    @Produces(MediaType.TEXT_PLAIN)
    @Secured("isAuthenticated()") // <1>
    @Get("/") // <3>
    String index(Principal principal) { // <4>
        principal.name
    }

    @Produces(MediaType.TEXT_PLAIN)
    @Secured(SecurityRule.IS_ANONYMOUS) // <1>
    @Get("/anonymous") // <3>
    String anonymous(Principal principal) { // <4>
        principal.name
    }
}

Would allow unauthenticated access for both endpoints

### Actual Behaviour

Returns 401 for both endpoints.

### Environment Information

- **Operating System**: MAC Os 10.13.6
- **Micronaut Version: 1.0.0.RC2
- **JDK Version: 8.0.172-zulu
