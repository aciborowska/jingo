**Saving multiple cookies with `cookies` method does not work.**

1. Create a sample app
2. Create a controller


```
@Requires(property = 'spec.name', value = 'CookiesSpec')
    @Controller("/cookies")
    static class CookiesController {

        @Produces(MediaType.TEXT_HTML)
        @Get
        HttpResponse index() {
            Cookie a = Cookie.of('A', 'B')
            Cookie c = Cookie.of('C', 'D')
            Set<Cookie> cookies = [a, c] as Set<Cookie>
            HttpResponse.ok('''
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>
<body>
Content of the document......
</body>
</html>
''').cookies(cookies)
        }
```
3. Visit `/cookies`

### Expected Behaviour

Multiple cookies are saved.

### Actual Behaviour

None is saved.

### Environment Information

- **Operating System**: MacOS 10.15.2
- **Micronaut Version:** 1.2.8
- **JDK Version:** 1.8
