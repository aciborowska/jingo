**Steps to Reproduce**
In a Micronaut application, add a property starting with 'z' and inject it into a bean with @Value

**Expected Behaviour**
The bean can be instantiated and the property's value is injected into it

**Actual Behaviour**
The bean fails to instantiate with the message:
Message: Error resolving field value [${z}]. Property doesn't exist or cannot be converted

**Environment Information**

    Operating System: Linux
    Micronaut Version: 1.0.2 (and Master-branch)
    JDK Version: 1.8.0_172

**Example Application**

    https://github.com/BartRobeyns/MicronautZProperty
