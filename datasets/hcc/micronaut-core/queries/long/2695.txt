For Kotlin projects, the Micronaut 1.3.0.RC1 `mn` cli does generate a `annotationProcessor` configuration in `build.gradle`, but it should generate `kapt` configuration.

### Steps to Reproduce

`mn create-app microtrends --features spock,graal-native-image-kotlin,data-jdbc --lang kotlin`

### Expected Behaviour

`kapt("io.micronaut.data:micronaut-data-processor")`

### Actual Behaviour

`annotationProcessor "io.micronaut.data:micronaut-data-processor"`

### Environment Information

- **Operating System**: macOS 10.15.2
- **Micronaut Version:** 1.3.0.RC1
- **JDK Version:** 1.8.0_202
