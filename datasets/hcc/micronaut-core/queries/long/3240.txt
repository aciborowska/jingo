### Steps to Reproduce

1. Clone repository https://github.com/ivangfr/graalvm-quarkus-micronaut-springboot
2. Navigate to `book-api/micronaut-book-api` folder
3. Run `./gradlew clean run`

### Expected Behaviour

As the app is configured in Gradle as Java 8, it should run in JDK 1.8

### Actual Behaviour

App doesn't run in JDK 1.8. In JDK 11 it does.
