**Describe the bug**
Searching for a draft product which appears in the product list returns no results if the [WooCommerce Product Search](https://woocommerce.com/products/woocommerce-product-search/) extension is enabled. 

I'm not sure if this is a bug with the app or with the extension, but wanted to point it out here since it's an extension WooCommerce sells. 

**To Reproduce**
Steps to reproduce the behavior:
1. Go to 'Products' and click on the search icon.
2. Click on 'search for any products in the list that are in draft mode
3. You can use filters or not, the draft products should display. 
4. Install and activate the WooCommerce Product Search extension and set it up.
5. Come back to the app and search for the same draft product.
4. See error with no results returning.
