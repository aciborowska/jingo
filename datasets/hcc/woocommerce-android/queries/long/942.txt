**Describe the bug**
With 1.5-rc-1 version it isn't possible to fulfill orders (change the order status from "processing" to "completed")

Reported in 1914823 zen

**To Reproduce**
Steps to reproduce the behavior:
1. Install 1.5-rc-1 on Android simulator (using an actual device should result in same results)
2. Log in, head to "Orders"
3. Select any order with "Processing" label/status
4. Click "Fulfill order"
5. Click "Mark order as complete"
6. Nothing happens.
