**Describe the bug**
I have a product with no start date for a sale and an end date set in wp-admin. In the app it shows today's date for the start and when editing the product it shows the end date for the start date.

**To Reproduce**
Steps to reproduce the behavior:
1. Edit a product in wp-admin and set a sale end date with no start date. 
2. Save the product.
3. Load the product in the app.
4. Review the from date in the product detail screen and in the edit details screen.

**Expected behavior**
There should be no date set for the start of the sale.
