With version 2.1, a simple HelloWorld code to eval distributed Set (**Map works well**)

```
    set.addItemListener(new ItemListener<Object>()
    {...
        public void itemAdded(ItemEvent<Object> item)
        {
            System.out.println("itemAdded=" + item.getItem() + "-" + set.size());
            for (Object object : set)
            {
                System.out.println(object);
            }
        }
    }, true);
```

Output:
- itemAdded=null-3
- item1
- item2
- item3
