`DistinctValuesAggregator` class returns a set of object for distinct values. Default hazelcast serializer does not support Set<T> so it serialize the result using java serializer.

`MultiAttributeProjection` returns Object[]. Default hazelcast serializer does not support Object[] so it serialize the result using java serializer.

This has two side effects:

* Non-java clients cannot support `DistinctValuesAggregator` and `MultiAttributeProjection`
* java client will use slow java serialization to deserialize the result
