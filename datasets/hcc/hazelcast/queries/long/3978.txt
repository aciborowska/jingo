Test1:

``` java
Hazelcast.newHazelcastInstance();
HazelcastInstance hz = HazelcastClient.newHazelcastClient();
IQueue<Object> queue = hz.getQueue("q");

Thread t = new Thread() {
    public void run() {
        try {
            queue.take();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
};
t.start();

Thread.sleep(5000);
t.interrupt();
t.join();

queue.offer("value");
Assert.assertNotNull(queue.poll());
```

Test2:

``` java
Hazelcast.newHazelcastInstance();
HazelcastInstance hz = HazelcastClient.newHazelcastClient();
ILock lock = hz.getLock("l");

lock.lock();

Thread t = new Thread() {
    public void run() {
        try {
            lock.lockInterruptibly();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
};
t.start();

Thread.sleep(5000);
t.interrupt();
t.join();

lock.unlock();
Assert.assertFalse(lock.isLocked());
```
