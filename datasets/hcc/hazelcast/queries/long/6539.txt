Documentation at http://docs.hazelcast.org/docs/3.5/manual/html-single/hazelcast-documentation.html#ssl is wrong:

ClientConfig config = new ClientConfig();
config.getSocketOptions().setSocketFactory( new SSLSocketFactory( props ) );

2 problems here:
a. config.getSocketOptions() is a deprecated API
b. there is no setSocketFactory(......) method

SSL on Client can not be configured using this documentation.
