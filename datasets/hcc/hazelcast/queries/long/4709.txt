Credits for reporting it go to Krishnakumar Kannan

Here is a reproducer:

``` java
    @Test
    public void testReadThroughShouldBackup() {
        String mapName = randomMapName();
        int noOfKeys = 100;

        Config config = new Config();
        MapStore mapLoader = createMapLoader(new AtomicInteger());
        MapStoreConfig mapStoreConfig = new MapStoreConfig()
                .setImplementation(mapLoader)
                .setInitialLoadMode(MapStoreConfig.InitialLoadMode.LAZY);
        config.getMapConfig(mapName).setMapStoreConfig(mapStoreConfig);
        HazelcastInstance[] hazelcastInstances = createHazelcastInstanceFactory(2).newInstances(config);

        IMap<Integer, Integer> map = hazelcastInstances[0].getMap(mapName);
        for (int i = 0; i < noOfKeys; i++) {
            map.get(i);
        }

        hazelcastInstances[1].shutdown();

        assertSizeEventually(noOfKeys, map);
    }
```
