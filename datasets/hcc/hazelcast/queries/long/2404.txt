After migration from 3.2 to 3.2.1 the following test started failing:

``` java
    @Test(expected = IllegalStateException.class)
    public void innerTransactionTest() {
        TransactionContext outerTnx = getInstance().newTransactionContext();
        outerTnx.beginTransaction();
        outerTnx.getMap("TEST1").put("TEST1", "TEST1");
        try {
            TransactionContext innerTnx = getInstance().newTransactionContext();
            innerTnx.beginTransaction();
            innerTnx.getMap("TEST2").put("TEST2", "TEST2");
            innerTnx.commitTransaction();
        } finally {
            outerTnx.rollbackTransaction();
        }
    }
```
