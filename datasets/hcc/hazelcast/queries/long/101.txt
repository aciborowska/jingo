Possible solution is to call lifecycleService.es.shutdown() after calling connectionManager.shutdown() [~line 94]. (or create a new method in LifecycleServiceClientImpl to do this b/c es is private)

public static void main(String[] args) {
    ClientConfig config = new ClientConfig();
    config.addAddress("192.168.1.115:5701");
    config.setGroupConfig(new GroupConfig("dev", "password"));
    HazelcastClient client = null;
    try {
        client = HazelcastClient.newHazelcastClient(config);
    } catch (Exception e) {
        e.printStackTrace();
    }
    if (client != null) {
        client.shutdown();
    }
}
