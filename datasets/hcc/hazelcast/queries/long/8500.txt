hot-restart tag cannot resolve placeholder for value at both "enabled" and "fsync".
e.g.: `<hz:hot-restart enabled="${hazelcast.hot-restart}" fsync="${hazelcast.hot-restart.fsync}"/>`
will throw:
org.springframework.beans.factory.xml.XmlBeanDefinitionStoreException: Line 212 in XML document from class path resource [hz.xml] is invalid; nested exception is org.xml.sax.SAXParseException; lineNumber: 212; columnNumber: 76; cvc-datatype-valid.1.2.1: '${hazelcast.hot-restart.fsync}' is not a valid value for 'boolean'.
        at org.springframework.beans.factory.xml.XmlBeanDefinitionReader.doLoadBeanDefinitions(XmlBeanDefinitionReader.java:399)
