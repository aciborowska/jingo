We're on hazelcast 3.2.1.

When I use iterator.remove(), it doesn't remove an item from the queue.

I get an IllegalStateException when I don't call 'iterator.next()' prior to 'iterator.remove()' - as I should, so iterator.remove() is doing something - just not removing my item.

Here's my test:
    @Test
    public void testQueueIteratorRemove()  {

```
    IQueue<String> queue = hzInstance.getQueue("testQueue");
    queue.add("one");
    queue.add("two");
    queue.add("three");
    queue.add("four");

    assertEquals(4, queue.size());
    // queue.remove() works fine:
    queue.remove("four");
    assertEquals(3, queue.size());

    Iterator<String> iterator = queue.iterator();

    while(iterator.hasNext()) {
        iterator.next();
        // doesn't remove:
        iterator.remove();
    }
    assertEquals(0, queue.size());

}
```
