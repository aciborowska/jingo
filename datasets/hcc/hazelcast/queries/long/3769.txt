There is a problem with client authentication. Instead of getting an AuthenticationException, a IllegalStateException is thrown. I see that the AuthenticationRequest does throw an exception; but somehow it leads to the connection getting closed and the client not being able to read out the exception.

```
@Test(expected = AuthenticationException.class)
    public void authenticationError() throws Throwable {
        final HazelcastInstance hzlCluster = Hazelcast.newHazelcastInstance();

        final ClientConfig clientConfig = new ClientConfig();
        clientConfig.setGroupConfig(new GroupConfig("unknown-group", "invalid-pwd"));

        final List<String> addresse = new ArrayList<String>();
        addresse.add("localhost:" + hzlCluster.getConfig().getNetworkConfig().getPort());
        clientConfig.setAddresses(addresse);

        try {
            HazelcastClient.newHazelcastClient(clientConfig);
        } catch (final RuntimeException e) {
            if (e.getCause() != null && e.getCause() instanceof AuthenticationException) {
                throw e.getCause();
            } else {
                throw e;
            }
        }
    }
```
