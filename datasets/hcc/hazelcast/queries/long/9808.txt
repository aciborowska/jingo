A client cannot add index to map using Portable values without registering Portable classes on member.

Below test fails

```
    @Test
    public void test_addIndexFromClient_portable() {
        Config config = new Config();
        HazelcastInstance hz1 = factory.newHazelcastInstance(config);

        ClientConfig clientConfig = new ClientConfig();
        clientConfig.getSerializationConfig()
                    .addPortableFactory(TestSerializationConstants.PORTABLE_FACTORY_ID, new PortableFactory() {
                        public Portable create(int classId) {
                            return new NamedPortable();
                        }

                    });
        HazelcastInstance client = factory.newHazelcastClient(clientConfig);
        IMap<String, NamedPortable> test = client.getMap("test");

        test.put("key1", new NamedPortable("name-1", 1));
        test.addIndex("name", true);
    }

```
