I create simple application for this issue: https://github.com/samuelbr/hazelcast-multimapbug.
1. Checkout project.
2. Make project using: _mvn clean package_
3. In target folder run: _java -jar hz.multimapbug-0.0.1-SNAPSHOT.jar_
4. In output you can see:

```

        at com.example.hz.multimapbug.App.main(App.java:27)
```
