The Javascript popup prompt is incorrect for numeric fields.  For example, if you run up the petclinic demo and attempt to enter a new Owner, the popup for Telephone says "Enter City (required)".
Here is the underlying div:
<div id="roo_owner_telephone"><label for="_telephone">Telephone:</label><input id="_telephone" name="telephone" style="width:250px" type="text" value="" size="0" maxlength="30"/><br /><script type="text/javascript">Spring.addDecoration(new Spring.ElementDecoration({elementId : "_telephone", widgetType : "dijit.form.ValidationTextBox", widgetAttrs : {promptMessage: "Enter City (required)", invalidMessage: "field.valid.integer", regExp: "-?[0-9]*", required : true}})); </script></div>
As you can see the promptMessage is wrong.
