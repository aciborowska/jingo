The file samples/openid-xml/src/main/webapp/js/openid-jquery.js should replace the URL http://yahoo.com/ with https://me.yahoo.com/. The correct URL is used in samples/openid-jc/src/main/resources/resources/js/openid-jquery.js.
Without this fix, the XML application fails to use OpenID against Yahoo!, otherwise you get this stack:
