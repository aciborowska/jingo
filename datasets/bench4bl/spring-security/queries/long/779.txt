Calling the attemptAuthentication method of the org.springframework.security.providers.rcp.RemoteAuthenticationManager interface throws an IllegalArgumentException when Hessian attempts to deserialize a GrantedAuthorityImpl object.
If the object does not have a no-argument constructor, Hessian calls another constructor with null, 0 or false as the arguments.  An Assert in the GrantedAuthorityImpl constructor throws IllegalArgumentException if the passed argument is null.
This issue is related to issue SEC-123.
