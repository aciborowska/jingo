spring security config : < remember-me / > yields an error "Factory bean 'org.springframework.security.userDetailsServiceFactory' not found".
this can be reproduced by import spring-security-samples-tutorial-3.1.0.M2.war from spring 3.1.0.M2 distribution on a freshly installed STS 2.5.1 installer from the spring website. After import -> war, add spring nature and enable "Load NameSpaceHandlers and XSDs from projects classpath" from project properties.
