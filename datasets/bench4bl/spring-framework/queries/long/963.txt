When executing JdbcTemplate queryForRowSet method, a NoClassDefFoundError excpetion is thrown for javax.sql.rowset.CachedRowSet, which is part of Java 1.5.  The javadoc for the method states that results will be mapped to an SqlRowSet which is a wrapper class for javax.sql.RowSet.  However, it also implies that it could possibly wrap javax.sql.rowset.CachedRowSet, which seems to be the case.
